Welcome to the MUD Server Documentation

This document provides an overview of the MUD Server and its features.

Health Endpoint

The health endpoint allows you to check the status of the server. You can access it via the following URL:

http://localhost:8080/api/health

When accessed, it returns a simple message indicating whether the server is up and running.

Features

User Management

The MUD Server includes robust user management features, allowing you to create, update, and delete user accounts.

Game Rooms

MUD Server supports multiple game rooms where players can interact. Each room has its own set of rules and content.

Chat System

A built-in chat system enables real-time communication between players within the same game room.
